# Rift-OS

Rift-OS は「数秒でサクッと起動する」ことを目標にした、超軽量な次世代 OS プロトタイプです。低レイヤ最適化と無駄排除に注力し、ベアメタル環境での高速起動・低レイテンシを実現するための実験的実装を提供します。

## 主要特徴
- 超軽量タスク制御（Sleep）：ミリ秒未満の高速復帰
- 高速ファイル I/O（MMAP ベースの CreateFile 相当）
- ベアメタル優先設計：クロスコンパイルとカスタムブートサポート
- 小さなランタイムと最小依存

## 目標
- ブート時間の大幅短縮
- 組込み / エッジ用途での低レイテンシ動作
- 小規模なコードベースで探索的な OS 機構を検証
